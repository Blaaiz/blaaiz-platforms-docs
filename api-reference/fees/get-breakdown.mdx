---
title: Get fees breakdown
description: Calculate fees for a transaction before initiating it
api: POST /api/external/fees
---

Get a detailed breakdown of fees for a payout or collection transaction before initiating it. This helps you understand the total cost and what the recipient will receive.

## Headers

<ParamField header="x-blaaiz-api-key" type="string" required>
  Your API key for authentication
</ParamField>

## Body Parameters

<ParamField body="from_currency_id" type="string" required>
  The currency code you're sending from (e.g., NGN, CAD, USD)
</ParamField>

<ParamField body="to_currency_id" type="string" required>
  The currency code the recipient will receive (e.g., NGN, CAD, USD)
</ParamField>

<ParamField body="from_amount" type="number">
  The amount you want to send from your wallet. Use this when you know how much you want to deduct from your balance.
</ParamField>

<ParamField body="to_amount" type="number">
  The exact amount you want the recipient to receive. Use this when you want to ensure the recipient gets a specific amount after fees.
</ParamField>

<ParamField body="method" type="string" required>
  The transaction method. Options: `bank_transfer`, `interac`, `card`, `open_banking`
</ParamField>

<Note>
You must provide either `from_amount` or `to_amount`, but not both.
</Note>

## Response

<ResponseField name="status" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  Response message
</ResponseField>

<ResponseField name="data" type="object">
  The fee breakdown details
  
  <Expandable title="Fee breakdown object">
    <ResponseField name="you_send" type="number">
      Total amount that will be deducted from your wallet (includes all fees)
    </ResponseField>
    
    <ResponseField name="total_amount_to_send" type="number">
      The base amount being sent (before recipient fees)
    </ResponseField>
    
    <ResponseField name="recipient_gets" type="number">
      The final amount the recipient will receive after all fees
    </ResponseField>
    
    <ResponseField name="total_fees" type="number">
      Total fees charged for the transaction
    </ResponseField>
    
    <ResponseField name="platform_fee" type="number">
      Blaaiz platform fee
    </ResponseField>
    
    <ResponseField name="method_fee" type="number">
      Fee specific to the payment method
    </ResponseField>
    
    <ResponseField name="exchange_rate" type="number">
      Exchange rate applied (for cross-currency transactions)
    </ResponseField>
    
    <ResponseField name="from_currency" type="string">
      The source currency
    </ResponseField>
    
    <ResponseField name="to_currency" type="string">
      The destination currency
    </ResponseField>
    
    <ResponseField name="method" type="string">
      The payment method
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL - Same Currency

  --url https://api-dev.blaaiz.com/api/external/fees \
  --header 'Content-Type: application/json' \
  --header 'x-blaaiz-api-key: YOUR_API_KEY' \
  --data '{
    "from_currency_id": "NGN",
    "to_currency_id": "NGN",
    "from_amount": 50000,
    "method": "bank_transfer"
  }'
```

```python Python
import requests

url = "https://api-dev.blaaiz.com/api/external/fees"
headers = {
    "Content-Type": "application/json",
    "x-blaaiz-api-key": "YOUR_API_KEY"
}
payload = {
    "from_currency_id": "NGN",
    "to_currency_id": "NGN",
    "from_amount": 50000,
    "method": "bank_transfer"
}

response = requests.post(url, json=payload, headers=headers)
data = response.json()
```
