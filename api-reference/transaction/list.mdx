---
title: List transactions
description: Retrieve a paginated list of transactions with filtering options.
api: POST /api/external/transaction
---

This endpoint allows you to retrieve a paginated list of transactions for the business with filtering options.

## Endpoint

```
POST /api/external/transaction
```

## Headers

<ParamField header="Accept" type="string" default="application/json">
  Content type header.
</ParamField>

<ParamField header="x-blaaiz-api-key" type="string" required>
  Your API key for authentication.
</ParamField>

## Body parameters

<ParamField body="start_date" type="string">
  Start date for filtering (YYYY-MM-DD format).
</ParamField>

<ParamField body="end_date" type="string">
  End date for filtering (YYYY-MM-DD format).
</ParamField>

<ParamField body="wallet_id" type="string">
  Filter by wallet ID.
</ParamField>

<ParamField body="type" type="string">
  Transaction type: `SEND_MONEY` or `FUND_WALLET`.
</ParamField>

<ParamField body="status" type="string">
  Transaction status: `FAILED` or `SUCCESSFUL`.
</ParamField>

<ParamField body="customer_id" type="string">
  Filter by customer ID.
</ParamField>

```python Python
import requests

url = "https://api-dev.blaaiz.com/api/external/transaction"
headers = {
    "Accept": "application/json",
    "Content-Type": "application/json",

}

    "start_date": "2025-01-01",
    "end_date": "2025-12-31",
    "status": "SUCCESSFUL"
}

data = response.json()
```
