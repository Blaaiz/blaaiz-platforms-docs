---
title: Resolve bank account
description: Verify and resolve bank account details
api: POST /api/external/bank/account-lookup
---

This endpoint allows you to verify the validity of a bank account by providing the account number and the associated bank ID. It returns information about the account, such as the account holder's name, to ensure accuracy before initiating transactions.

We advise using this to confirm recipient details for payments, transfers, or any transaction involving a bank account.

**Example Use Cases:**

- Validate a recipient's bank account details during payment setup.
- Ensure accurate data entry before processing a transaction.
- Provide real-time verification for user onboarding or payout processes.

## Headers

<ParamField header="x-blaaiz-api-key" type="string" required>
  Your API key for authentication
</ParamField>

## Body Parameters

<ParamField body="account_number" type="string" required>
  The bank account number to verify
</ParamField>

<ParamField body="bank_code" type="string" required>
  The bank code (can be retrieved from the List Banks endpoint)
</ParamField>

<ParamField body="currency" type="string" required>
  The currency of the account (e.g., NGN, USD)
</ParamField>

## Response

<ResponseField name="status" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  Response message
</ResponseField>

<ResponseField name="data" type="object">
  The resolved account details
  
  <Expandable title="Account details">
    <ResponseField name="account_number" type="string">
      The verified account number
    </ResponseField>
    
    <ResponseField name="account_name" type="string">
      The name of the account holder
    </ResponseField>
    
    <ResponseField name="bank_code" type="string">
      The bank code
    </ResponseField>
    
    <ResponseField name="bank_name" type="string">
      The name of the bank
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl --request POST \
  --url https://api-dev.blaaiz.com/api/external/bank/account-lookup \
  --header 'Content-Type: application/json' \
  --header 'x-blaaiz-api-key: YOUR_API_KEY' \
  --data '{
    "account_number": "0123456789",
    "bank_code": "044",
    "currency": "NGN"
  }'
```

```javascript Node.js
const response = await fetch('https://api-dev.blaaiz.com/api/external/bank/account-lookup', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-blaaiz-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    account_number: '0123456789',
    bank_code: '044',
    currency: 'NGN'
  })
});

const data = await response.json();
```

```python Python
import requests

url = "https://api-dev.blaaiz.com/api/external/bank/account-lookup"
headers = {
    "Content-Type": "application/json",
    "x-blaaiz-api-key": "YOUR_API_KEY"
}
payload = {
    "account_number": "0123456789",
    "bank_code": "044",
    "currency": "NGN"
}

response = requests.post(url, json=payload, headers=headers)
data = response.json()
```
</RequestExample>
