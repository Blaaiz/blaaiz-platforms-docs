---
title: List all banks
description: Retrieve a list of all supported banks
api: GET /api/external/bank
---

Retrieve a list of all banks supported by Blaaiz for payouts and collections.

## Headers

<ParamField header="x-blaaiz-api-key" type="string" required>
  Your API key for authentication
</ParamField>

## Query Parameters

<ParamField query="currency" type="string">
  Filter banks by currency code (e.g., NGN, USD, CAD)
</ParamField>

<ParamField query="country" type="string">
  Filter banks by country code (e.g., NG, US, CA)
</ParamField>

## Response

<ResponseField name="status" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  Response message
</ResponseField>

<ResponseField name="data" type="array">
  Array of bank objects
  
  <Expandable title="Bank object">
    <ResponseField name="id" type="string">
      Unique identifier for the bank
    </ResponseField>
    
    <ResponseField name="name" type="string">
      The name of the bank
    </ResponseField>
    
    <ResponseField name="code" type="string">
      The bank code
    </ResponseField>
    
    <ResponseField name="country" type="string">
      Country code where the bank operates
    </ResponseField>
    
    <ResponseField name="currency" type="string">
      Currency supported by the bank
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl --request GET \
  --url 'https://api-dev.blaaiz.com/api/external/bank?currency=NGN' \
  --header 'x-blaaiz-api-key: YOUR_API_KEY'
```

```javascript Node.js
const response = await fetch('https://api-dev.blaaiz.com/api/external/bank?currency=NGN', {
  method: 'GET',
  headers: {
    'x-blaaiz-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

url = "https://api-dev.blaaiz.com/api/external/bank"
headers = {
    "x-blaaiz-api-key": "YOUR_API_KEY"
}
params = {
    "currency": "NGN"
}

response = requests.get(url, headers=headers, params=params)
data = response.json()
```
</RequestExample>
