---
title: Get identification type
description: Retrieve identification types for virtual bank account creation
api: GET /api/external/virtual-bank-account/identification-type
---

Retrieve the available identification types for virtual bank account creation. This endpoint helps you determine which identification documents are required based on customer type and country.

## Headers

<ParamField header="x-blaaiz-api-key" type="string" required>
  Your API key for authentication
</ParamField>

## Query Parameters

<ParamField query="customer_id" type="string">
  The unique identifier of the customer. If provided, country and type are not required.
</ParamField>

<ParamField query="country" type="string">
  Two-letter country code (e.g., NG). Required if customer_id is not provided.
</ParamField>

<ParamField query="type" type="string">
  The type of customer. Possible values: `business`, `individual`.
</ParamField>
