---
title: Get webhook URLs
description: Retrieve your registered webhook URLs
api: GET /api/external/webhook
---

Retrieve the currently registered webhook URLs for your account.

## Headers

<ParamField header="x-blaaiz-api-key" type="string" required>
  Your API key for authentication
</ParamField>

## Response

<ResponseField name="status" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  Response message
</ResponseField>

<ResponseField name="data" type="object">
  The webhook configuration
  
  <Expandable title="Webhook configuration">
    <ResponseField name="payout_url" type="string">
      Registered payout webhook URL
    </ResponseField>
    
    <ResponseField name="collection_url" type="string">
      Registered collection webhook URL
    </ResponseField>
    
    <ResponseField name="customer_url" type="string">
      Registered customer webhook URL
    </ResponseField>
    
    <ResponseField name="virtual_account_url" type="string">
      Registered virtual account webhook URL
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      Timestamp when webhooks were first configured
    </ResponseField>
    
    <ResponseField name="updated_at" type="string">
      Timestamp when webhooks were last updated
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl --request GET \
  --url https://api-dev.blaaiz.com/api/external/webhook \
  --header 'x-blaaiz-api-key: YOUR_API_KEY'
```

```javascript Node.js
const response = await fetch('https://api-dev.blaaiz.com/api/external/webhook', {
  method: 'GET',
  headers: {
    'x-blaaiz-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

url = "https://api-dev.blaaiz.com/api/external/webhook"
headers = {
    "x-blaaiz-api-key": "YOUR_API_KEY"
}

response = requests.get(url, headers=headers)
data = response.json()
```
</RequestExample>
