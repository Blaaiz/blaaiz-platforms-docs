---
title: Update webhook URL
description: Update your registered webhook URLs.
---

This endpoint is used to update your registered webhook URLs.

## Endpoint

```
PUT /api/external/webhook
```

## Headers

<ParamField header="Accept" type="string" default="application/json">
  Content type header.
</ParamField>

<ParamField header="x-blaaiz-api-key" type="string" required>
  Your API key for authentication.
</ParamField>

## Body parameters

<ParamField body="collection_url" type="string">
  The URL for collection webhooks.
</ParamField>

<ParamField body="payout_url" type="string">
  The URL for payout webhooks.
</ParamField>

## Response

<ResponseField name="message" type="string">
  Success message.
</ResponseField>

<RequestExample>
```bash cURL
curl --request PUT \
  --url https://api-dev.blaaiz.com/api/external/webhook \
  --header 'Accept: application/json' \
  --header 'Content-Type: application/json' \
  --header 'x-blaaiz-api-key: YOUR_API_KEY' \
  --data '{
    "collection_url": "https://yourapp.com/webhooks/collection",
    "payout_url": "https://yourapp.com/webhooks/payout"
  }'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "message": "Webhook record updated successfully"
}
```
</ResponseExample>
