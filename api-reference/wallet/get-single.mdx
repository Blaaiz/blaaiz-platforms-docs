---
title: Get wallet
description: Retrieve a single wallet by ID.
api: GET /api/external/wallet/{id}
---

This endpoint retrieves a single wallet.

## Endpoint

```
GET /api/external/wallet/{id}
```

## Path parameters

<ParamField path="id" type="string" required>
  The unique identifier of the wallet.
</ParamField>

## Headers

<ParamField header="Accept" type="string" default="application/json">
  Content type header.
</ParamField>

<ParamField header="x-blaaiz-api-key" type="string" required>
  Your API key for authentication.
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the wallet.
</ResponseField>

<ResponseField name="business_id" type="string">
  Business identifier.
</ResponseField>

<ResponseField name="currency" type="string">
  Currency code (e.g., NGN, CAD, USD).
</ResponseField>

<ResponseField name="currency_id" type="string">
  Currency identifier.
</ResponseField>

<ResponseField name="amount" type="number">
  Current wallet balance.
</ResponseField>

<ResponseField name="is_active" type="boolean">
  Whether the wallet is active.
</ResponseField>
