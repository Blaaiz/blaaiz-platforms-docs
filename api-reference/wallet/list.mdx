---
title: List all wallets
description: Retrieve all business wallets
api: GET /api/external/wallet
---

This endpoint retrieves a list of wallets. The response includes details for each wallet, such as the id, currency, and balance.

## Headers

<ParamField header="x-blaaiz-api-key" type="string" required>
  Your API key for authentication
</ParamField>

## Response

<ResponseField name="status" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  Response message
</ResponseField>

<ResponseField name="data" type="array">
  Array of wallet objects
  
  <Expandable title="Wallet object">
    <ResponseField name="id" type="string">
      Unique identifier for the wallet
    </ResponseField>
    
    <ResponseField name="currency" type="string">
      The currency code for this wallet (e.g., NGN, CAD, USD)
    </ResponseField>
    
    <ResponseField name="balance" type="number">
      Current balance in the wallet
    </ResponseField>
    
    <ResponseField name="available_balance" type="number">
      Available balance (excluding pending transactions)
    </ResponseField>
    
    <ResponseField name="pending_balance" type="number">
      Balance from pending transactions
    </ResponseField>
    
    <ResponseField name="status" type="string">
      Status of the wallet (active, inactive, suspended)
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      Timestamp when the wallet was created
    </ResponseField>
    
    <ResponseField name="updated_at" type="string">
      Timestamp when the wallet was last updated
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl --request GET \
  --url https://api-dev.blaaiz.com/api/external/wallet \
  --header 'x-blaaiz-api-key: YOUR_API_KEY'
```

```javascript Node.js
const response = await fetch('https://api-dev.blaaiz.com/api/external/wallet', {
  method: 'GET',
  headers: {
    'x-blaaiz-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

url = "https://api-dev.blaaiz.com/api/external/wallet"
headers = {
    "x-blaaiz-api-key": "YOUR_API_KEY"
}

response = requests.get(url, headers=headers)
data = response.json()
```
