---
title: Attach customer to collection
description: Associate a transaction (collection) with a customer.
api: POST /api/external/collection/attach-customer
---

This endpoint allows you to associate a transaction (collection) with a customer. This way, the transaction will be registered against the customer.

<Note>
Only collections can be updated like this.
</Note>

## Endpoint

```
POST /api/external/collection/attach-customer
```

## Headers

<ParamField header="Accept" type="string" default="application/json">
  Content type header.
</ParamField>

<ParamField header="x-blaaiz-api-key" type="string" required>
  Your API key for authentication.
</ParamField>

## Body parameters

<ParamField body="customer_id" type="string" required>
  The unique identifier of the customer.
</ParamField>

<ParamField body="transaction_id" type="string" required>
  The unique identifier of the transaction.
</ParamField>

## Response

<ResponseField name="message" type="string">
  Success message.
</ResponseField>

<RequestExample>
```bash cURL
curl --request POST \
  --url https://api-dev.blaaiz.com/api/external/collection/attach-customer \
  --header 'Accept: application/json' \
  --header 'Content-Type: application/json' \
  --header 'x-blaaiz-api-key: YOUR_API_KEY' \
  --data '{
    "customer_id": "9d4c4ec5-59ea-4130-bf8a-6a5edec401ee",
    "transaction_id": "fb87c0f8-31e3-46f3-b0d3-1ff2618565bc"
  }'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "message": "Transaction updated successfully."
}
```
</ResponseExample>
