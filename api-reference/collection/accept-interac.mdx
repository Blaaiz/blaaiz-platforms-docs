---
title: Accept Interac money request
description: Accept an Interac transfer into your business wallet.
api: POST /api/external/accept-interac-money-request
---

This endpoint allows you to accept an Interac transfer into your business wallet.

## Endpoint

```
POST /api/external/accept-interac-money-request
```

## Headers

<ParamField header="Accept" type="string" default="application/json">
  Content type header.
</ParamField>

<ParamField header="x-blaaiz-api-key" type="string" required>
  Your API key for authentication.
</ParamField>

## Body parameters

<ParamField body="reference_number" type="string" required>
  Interac reference number.
</ParamField>

<ParamField body="security_answer" type="string" required>
  Security answer for the Interac transfer.
</ParamField>

<ParamField body="email" type="string">
  Email address associated with the Interac transfer.
</ParamField>

## Response

<ResponseField name="message" type="string">
  Success message.
</ResponseField>

<ResponseField name="transaction_id" type="string">
  Unique transaction identifier.
</ResponseField>

<ResponseField name="url" type="string">
  URL for additional information (if applicable).
</ResponseField>

<RequestExample>
```bash cURL
curl --request POST \
  --url https://api-dev.blaaiz.com/api/external/accept-interac-money-request \
  --header 'Accept: application/json' \
  --header 'Content-Type: application/json' \
  --header 'x-blaaiz-api-key: YOUR_API_KEY' \
  --data '{
    "reference_number": "ABC123456",
    "security_answer": "MySecretAnswer"
  }'
```
</RequestExample>
