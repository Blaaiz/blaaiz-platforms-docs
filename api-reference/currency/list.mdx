---
title: List all currencies
description: Retrieve all supported currencies
api: GET /api/external/currency
---

This endpoint retrieves a list of currencies. The response includes details for each currency, such as the id, name, code, and status.

## Headers

<ParamField header="x-blaaiz-api-key" type="string" required>
  Your API key for authentication
</ParamField>

## Response

<ResponseField name="status" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  Response message
</ResponseField>

<ResponseField name="data" type="array">
  Array of currency objects
  
  <Expandable title="Currency object">
    <ResponseField name="id" type="string">
      Unique identifier for the currency
    </ResponseField>
    
    <ResponseField name="code" type="string">
      The currency code (e.g., NGN, CAD, USD, EUR, GBP)
    </ResponseField>
    
    <ResponseField name="name" type="string">
      The full name of the currency
    </ResponseField>
    
    <ResponseField name="symbol" type="string">
      The currency symbol (e.g., ₦, $, €, £)
    </ResponseField>
    
    <ResponseField name="country" type="string">
      The country or region where the currency is used
    </ResponseField>
    
    <ResponseField name="supported_methods" type="array">
      Array of supported payment methods for this currency
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl --request GET \
  --url https://api-dev.blaaiz.com/api/external/currency \
  --header 'x-blaaiz-api-key: YOUR_API_KEY'
```

```javascript Node.js
const response = await fetch('https://api-dev.blaaiz.com/api/external/currency', {
  method: 'GET',
  headers: {
    'x-blaaiz-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

url = "https://api-dev.blaaiz.com/api/external/currency"
headers = {
    "x-blaaiz-api-key": "YOUR_API_KEY"
}

response = requests.get(url, headers=headers)
data = response.json()
```
</RequestExample>
